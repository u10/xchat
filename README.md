# XChat

> This is a IM WebApp use Vue.js.

[online demo](http://vaniship-xchat.daoapp.io/)

## Install
``` bash
npm install -g u10/xchat

xchat
```

## Build

``` bash
git clone https://github.com/u10/xchat.git

cd xchat

npm install

npm run build

./dist/bin/xchat
```

## Development

``` bash
git clone https://github.com/u10/xchat.git

cd xchat

npm install

npm run coffee

# start server
./bin/xchat

# serve with hot reload at localhost:8080
npm run dev
```
